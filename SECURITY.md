# Security Policy

## Scope

GenAI Repo Auditor is intended for defensive security review of repositories that the operator owns or is explicitly authorized to assess.

Do not use this project for:

- unauthorized scanning
- production or staging probing
- credential access
- exploitation of third-party systems
- weaponized exploit generation
- disclosure of full secret values

## Reporting vulnerabilities in this project

Report vulnerabilities in GenAI Repo Auditor through GitHub's private vulnerability reporting if enabled, or by contacting the repository maintainer directly.

Do not create a public GitHub Issue for vulnerabilities involving:

- credentials
- tokens
- customer data
- exploitable security bugs
- private repository contents
- audit findings from third-party repositories

## Audit findings produced by this tool

Findings generated against a target repository may contain sensitive information. Review them locally before creating GitHub Issues or publishing any details.

Public repository Issue creation is intentionally guarded. Use public Issues only when disclosure is approved.
